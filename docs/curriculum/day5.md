# Day5（7/15）ログイン・アカウント管理・テスト（Day5コード使用）

## 目的（Day5でできるようになること）
- ログイン機能を理解できる
- 管理者でアカウント管理ができる
- テストを実行できる

## 使用するコード
`stages/day5` を使う（完成版）

---

## 1. 起動
```bash
cd stages/day5
mvn spring-boot:run
```

## 2. ログイン
```
http://localhost:8080/login
```
初期ユーザー:
- admin / admin123
- user1 / password

---

## 3. アカウント管理（管理者）
```
http://localhost:8080/users
```
新規作成 / 編集 / 削除を確認する。

---

## 4. 管理者の勤怠編集
```
http://localhost:8080/admin/attendances
```
任意の勤怠を編集し、時刻と状態が更新されることを確認。

---

## 5. テスト
```bash
mvn test
```

---

## 6. チェックポイント
- ログインできる
- 管理者機能が使える
- テストが通る

---

## 7. 時間割目安
- 午前: ログイン/管理機能(90分)
- 午後: テスト(30分) / まとめ(30分)

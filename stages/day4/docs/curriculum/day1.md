# Day1（7/9）環境準備 + Java超入門 + 勤怠アプリ起動

## 目的（Day1でできるようになること）
- Java/ Maven の役割が分かる
- Spring Boot アプリを起動できる
- ログインして勤怠トップ画面を表示できる

## 業務ストーリー（今日の作業イメージ）
1. 勤怠システムを起動する  
2. ログインして「出勤/退勤」ができる状態を作る  

---

## 1. 前提（準備済みか確認）
- Java 17 / Maven 3.9+ / VS Code
- `mvn -version` が通る

---

## 2. Javaとは何か / JDK・JRE・JVM（超入門）
- Javaは「書いたコードをJVMで動かす」言語
- JDK = 開発キット（`javac` が入っている）
- JRE = 実行環境（JVM + ライブラリ）
- JVM = Javaの実行エンジン

---

## 3. Mavenとは何か（超入門）
- Mavenは「依存関係の管理」と「ビルド」を自動化するツール
- 使うコマンドはこの研修では2つだけ:
  - `mvn spring-boot:run`（起動）
  - `mvn test`（テスト）

---

## 4. アプリ起動
1. ルートで実行
```powershell
mvn spring-boot:run
```

2. ブラウザでログイン画面
```
http://localhost:8080/login
```

3. 初期アカウントでログイン
- admin / admin123（管理者）
- user1 / password（一般）

---

## 5. 勤怠トップ画面の確認
ログイン後トップ画面で以下を確認:
- 今日の状態（未出勤/出勤中/退勤済み）
- 出勤ボタン or 退勤ボタン
- 出勤時刻 / 退勤時刻が表示される

---

## 6. 出勤 → 退勤（動作確認）
1. 「出勤」ボタンを押す  
2. 状態が「出勤中」に変わる  
3. 「退勤」ボタンを押す  
4. 状態が「退勤済み」になる  

---

## 7. チェックポイント
- アプリが起動できた
- ログインできた
- 出勤 → 退勤 ができた

---

## 8. つまずきポイント
- `mvn` が通らない → 環境変数を確認
- 画面が出ない → 起動ターミナルのエラーを見る
- ログインできない → 初期アカウントを確認

---

## 9. 時間割目安
- 午前: Java/Maven入門(60分) / 起動(60分)
- 午後: ログイン/出勤/退勤(60分) / まとめ(30分)
